# Oil & Gas Production Analysis

## Contexte
Analyse des volumes annuels de production pétrolière et gazière (2015–2025).

## Résultats
- Tendance baissière régulière 2015–2019.
- Reprise fragile 2020–2023.
- Chute critique en 2025 (-59%).

## Fichiers
- `production_growth.csv` → données annuelles.
- `oil and gas analysis.ipynb` → code Python.
- `dashboard.png` → visuel Power BI.
